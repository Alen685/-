# GitHub User Finder

Автор: Балашова Алёна

## Описание

Простое приложение для поиска пользователей GitHub. 
Избранные сохраняются в JSON.

## Как запустить

1. Установите Python (если нет)
2. Запустите файл: main.py

## Как использовать

- Введите имя пользователя → нажмите Поиск
- Нажмите "В избранное" чтобы сохранить
- Выберите пользователя в списке → Удалить

## Что реализовано

- Поиск через GitHub API
- Добавление в избранное
- Сохранение в favorites.json
- Проверка пустого поля
- Работа без интернета (ошибка)

### Пример 1: Успешный поиск


Ввод: torvalds
Результат:
Логин: torvalds
Имя: Linus Torvalds
Репозиториев: 7
Подписчиков: 215000
Ссылка: https://github.com/torvalds

### Пример 2: Пользователь не найден


Ввод: sdjhfksjdhf
Результат: Пользователь 'sdjhfksjdhf' не найден

### Пример 3: Пустое поле



Ввод: (ничего)
Результат: Всплывает окно "Поле поиска не может быть пустым!"
